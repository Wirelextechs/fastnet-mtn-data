Authentication

Last updated August 23rd, 2025

To facilitate integration with Hubnet APIs, the generation of API keys is required for authentication. Access to each API necessitates the creation of specific keys tailored to its functionalities.

Authentication Methods

Hubnet APIs support various authentication methods to ensure secure access and interaction with the services. Below are the primary methods available:

  1. -> Bearer Token

token : Bearer { token }

Bearer Token

The easiest way to authenticate is by using Bearer Token. This involves including the token in the HTTP Authorization header. See below for a sample header request information.

HTTP

Copy 
 token : Bearer {token}

To obtain a Bearer Token, visit here, click on Create New API Keys ; token will be generated for you.
